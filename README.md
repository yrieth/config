A simple config reader.
It can add, remove keys and modify, get values.
The .conf files are alphabetically sorted by keys.

.conf format: <br />
  key <br />
  value <br />

.conf example: <br />
  a <br />
  val <br />
  b <br />
  val1 <br />
  c <br />
  val <br />
  d <br />
  val <br />
  e <br />
  val <br />
  f <br />
  val <br />
  size <br />
  50 <br />
